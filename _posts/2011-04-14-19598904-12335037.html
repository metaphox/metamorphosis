--- title: 完全不懂编程，该如何学习 Java？ layout: post --- 完全不懂编程，为什么要学习 Java？建议从 Python 或者 Schema 开始。两者都试试，看哪个你更喜欢。

Java 是学院派的、工业化的、引领上世纪九十年代语言审美观念风潮的、由一群坐在鸡笼般格子间里穿着衬衣打着领带面无表情由文档和死线驱动随时可以被解雇的码农们使用的语言。在中国的话甚至可以省去穿着衬衣打着领带这个条件。

是的，你可以用它开发 Android 或者 GWT ，不过任何学好 Python 的人都可以在一周里掌握相关的 Java 技能。更不用提那些手眼通天的 C++ 程序员，看不出他在乱敲键盘还是编程的 Perl 程序员，或者以问“你用的语言可以做 < 填入某种奇技淫巧的编程术语 > 吗？”为乐的 Ruby 程序员。哦，还有那些以默默享受编程、bitching XCode 和数钱为乐的 Objective-C 程序员。

是的，Java 没有死，它只是渐渐不再引人注意。
