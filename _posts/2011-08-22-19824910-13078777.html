--- title: 如何理解 Photoshop 里色阶中的灰场？ layout: post --- 色阶是指 Level？就是 Ctrl + L 出来的对话框？

此时如果你选择某一颜色（确切来说，通道，channel）调节的话，就只是重新映射这一通道内的颜色从最淡（0）到最浓（255）的分布。比如输入从 0 到 128，输出从 0 到 255，就是把原来 0 到 128 的分布均匀扩展到 0 到 255 的范围内（原来的 129 到 255 被丢弃了）。如果输入从 0 到 255，输出从 0 到 128，就是把原来的颜色分布压缩到一半。

调节某一通道的时候，其他颜色（通道）不受影响，也就是说，如果 RGB 模式你把红色输出调到 0，那其他两种颜色就按照没有红色的状况叠加。假如原图是纯白，此时就是纯绿加纯蓝等于青色（00FFFF）。
