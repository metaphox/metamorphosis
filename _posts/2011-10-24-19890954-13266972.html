--- title: Windows下德语键盘如何用中文输入法正确输入中文标点？ layout: post ---
  1. 下载 Microsoft Keyboard Layout Creator [1] 并安装
  2. 运行安装好的 MKLC，选择File -> load existing keyboard...
  3. 在出现的键盘布局列表中 "German"，确认
  4. 现在德文键盘布局出现在窗口中。点击 Backspace 左侧的 ´ （acute），在跳出的窗口中点击 All...
  5. 去掉右侧的两个 dead key 选项
  6. 将 <key> 后的 Unicode 代码删除，改为 \ （back slash），回车，它会自动被专为对应的Unicode 代码
  7. 保持 shift + <key> 后的 Unicode 代码不变
  8. 点击“ok”。如果警告多少个 Dead key 会因此失效，点击确定
  9. Project -> Properties... ，Name输入自己喜欢的名字，比如 KBDDEZH，Description 输入比如 German (for pinyin)
  10. Project -> Validate Layout，如果有 Warning，不必理会；如果有 Error，请看一下 log 文件找原因，一般是和现有布局重名
  11. Project -> Build DLL and Setup Package，会生成若干 DLL，拷贝 i386/KBDDEZH.DLL 到 system32 目录下
  12. 修改注册表 HKEY_LOCAL_MACHINE->SYSTEM->ControlSet001->Control->keyboard Layouts->E0220804 ，原本的 KBDUS.dll 或 KBDGR.DLL 为 KBDDEZH.DLL
  13. 重启

从此谷歌拼音输入法会使用德语键盘布局，并且在按下 Backspace 左侧（或 ß 右侧那个键）时输入顿号，shift ＋ 此键时输入居中圆点。代价是无法输入所有带 acute 或 grave 的词汇，比如 déjà vu 。
