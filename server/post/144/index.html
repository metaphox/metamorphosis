以时光的名义

按：以下的这些文字应该是2004年我在学校网站做事的时候写的。对于那段时间的记忆，现在已经非常模糊了，若不是在天边的blog上撞到它，我恐怕会就此忘记吧。感谢天边给俺留下了这个备份。我现在回想不起来，或者说理解不了，自己当初为什么要把它起名叫“以时光的名义”，以及我想要完成但是未完成的部分又是什么。

我猜我现在已经很少有精力和兴趣来写这样的文字了吧。大四时忽然觉得大学四年其实并没有什么太多值得纪念的，当然也有可能是因为可纪念的东西如此之多，以至于失去了分别纪念的意义。

真的，为什么要以时光的名义呢？

以时光的名义(未完成)

——sSky.cn 04"改版寄语

2004年4月12日晚，正在发烧头痛欲裂的lonelyfox接到boo的短信说，apple老大在团队交流里面贴下告示，号令ssky的工作人员对已颇显老态的学子天空页面做出调整。于是两天之后，boo将自己雪藏已久的改版方案和盘托出，我则祭起photoshop、illustrator与golive三把上古神兵，汇同擅长施展flash奥术的闪灵与宸忱，精通asp之道的bsonline，以及java魔法出神入化的Gale，六人一同踏上了改版的不归路……

史书记载，2001年夏末秋初，那个风起云涌、豪杰辈出的时代，几个大男孩凑在办公楼的两台破电脑前，也曾是像我们这样废寝忘食，昼夜奋战，搭出了ssky的第一块积木。三年一晃而过，开拓者的ID早已无人知晓（boo：拜托你看一看注册用户列表里最早的用户不就行了……），ssky却在接替者手中逐渐成长，不断完善，秉承了最初创立时的意愿，一直发展到了今天。在这三年间，ssky.org和ssky.cn两个域名，陪着多少学子度过了他们的大学时光？有多少人因ssky而结识？有多少故事缘起于ssky，又留痕于ssky？而像我这样，结识了ssky，结识了sskyer，并加入到ssky团队中来，并在ssky上实现自己不大不小的理想的人，又有多少？2004年五月三日，在这个阴冷的早晨，彻夜未眠的我抱着fox3000wang的枕头，坐在网站这台黑色的ibm Intelli Station前回想这一切的时候，由衷的为之感慨。

这一次学子改版，虽然由于人手和日程的安排问题而未免仓促，留下了大量的细节工作要做，但是改版的意图与方向，却的确是深思熟虑过的。必须感谢站长boo为这次改版所作出的精心策划，网站的所有板块和功能被精确的重新定位了，原有的首页被拆分，原先杂乱穿插排列的所有内容被划分到了四个大板块中：学生工作、勤工助学、社团和社区。学生工作与勤工助学实现的是学工办网站的基本职能；社团是新的板块，主要的目的是想要起到一个校区内社团组织的信息发布和交流平台的作用；而社区板块，也就是大家现在看到的，以新版论坛为核心，综合了若干娱乐功能的sSkyer社区页面，是我们这一次改版投入精力最大的部分。从论坛到社区，从简单到复杂，从粗略到细致，所有改变的初衷，无非为了sSky的发展。一个校园网站，最大的心愿是什么？就是希望能被在校园中生活过的，在论坛上灌过的人们记住，并且在哪怕离开学校后的很长一段时间里，这些人都能经常回来看看，聊聊。纵然渐渐不来了，或者站点没有了，也能在逐渐老去的人生中偶尔想起，成为他/她那段大学生活的精神纪念。要怎样才能产生这种效果？bbs上一定要聚集一群有建设性、幽默感和人文情怀的人。这是第一要素，没错。我们的学子天空，已经有很多这样的人了，但是还不够多。我们的愿望是，通过这一次改版，为sSky整合更强大的功能，突出展示、交流和互动的元素，凸现个性化、现代感和时效性，淡化官方形象，从而种下更多的梧桐树，引得更炫的凤凰来。为此我们重新设计了logo，就是大家现在随处可以见到的，一只展开的翅膀。寓意是很简单的，祝愿sSky.cn和所有sSkyer们都能拥有翅膀，如同社区首页左上角logo下面所写的一样，take wing, to the skys——乘风飞翔。（知道出处的朋友们就不要揭穿了……）为了造势，我们还想在改版之后进行一系列的宣传活动，印制我们自己的文化衫，组织一次sSkyer聚会……将来的将来，ssky还会有更多的发展，我们为之做出的，仅仅是奠定了一个并不成熟基础而已。更多的工作，包括我们想到和没有想到的，留给明天的sSkyer们完成。

至于为什么要换论坛？对网络编程有些了解的人应该都能明白，作为在校的学生，我们的水平有限，尚没有时间、精力和经验独立开发一个相对完整的论坛系统，只能采用网上广为流传的免费版本。这也就是有些用户说我们的bbs“像某个网站”的原因。论坛系统采用的动态网络编程语言种类决定了它需要何种数据库的后台支持，或者简单的说，论坛是php的，数据库就应该是mysql的。在01年网站最初成立的时候，论坛是雷傲的，后来转成了大家熟悉的Discuz!版，但是没有考虑到后期网站要发展，实现更多的功能，采用php的Discuz!作为简单的交流平台是够了，却不足以担负起更重的任务。这其中牵扯到很多相对比较深层的技术问题，而且也和最初的网站定位有关，毕竟当时的创立者应该是第一次做一个综合性的站点，缺乏经验和长期策划，仅仅是实现功能，未曾留有太多扩充的余地。我们这次改版更换论坛，是经过了周密的计划和反复论证，权衡了利弊之后才行动的。我也是2002年九月就开始在学子上潜水的，明白大家对旧论坛的感情，但是为了更好的发展，不得不作出取舍。在这里郑重地向大家道歉了，希望大家可以理解。还希望大家知道，sSky早期从雷傲转到Discuz!的那一次，丢了很多文物级的帖子。为了不重蹈覆辙，我们在四月末的几天里一直在进行数据转换测试。网上有一个discuz2dv的转换程序，但是只能转15000张帖子左右，更多就会出错误。为了解决这个问题，技术部的Gale连续几天彻夜未眠，详细摸透了Discuz!和动网的数据结构差异，用java语言完全原创了数据转换程序，86,300余张帖子，2万多ID数据（除去一小部分用了dvbbs过滤字符的id）无一在逃（报告帖子丢失是因为计数错误）。对于一个大二学生而言，这是了不起的成就。在此特别纪念他的为这次改版作出的贡献，作为同事、朋友和同班同学，我对他表示深深的敬意。

在论坛完成数据转换后的试运行的过程中，sSkyer们对新版论坛的反应是强烈的。有人用了悲伤和失望这样的字眼，一开始，这些词语同样让我觉得悲伤和失望。我不明白为什么从好的出发点用心去做的事情会获得这样的评论，但是过了8小时之后我开始明白，有这样的反应恰恰说明了学子天空在sSkyer心目中的地位。我想假如换了论坛和没换一样，波澜不兴，没有任何反响，恐怕才是我们最大的悲哀。现在的情况是，箭在弦上，不得不发。既然开始了，就尽量把它做好。当然相应的，也有很多热心的sSkyer们作出了中肯的正面评价，更有ray等前辈晓以经验道理，lonelyfox在这里也对他们表示由衷的感谢。同时，希望说了不想再来的sSkyer们千万不要真的离开ssky，论坛换了，人都还在啊，不想失去你们的身影，真的不想。
