事关爱情

中南大学校本部有一座正常学生每天都要出入一到三次的建筑，从来不去或者要去更多次就意味着你多少有些与众不同。这座面朝西侧，呈U字型的四层楼按它外墙上的几个金属字应唤作“饮食文化城”。必须承认这个名字比它的俗称，“大食堂”来得更准确一些，因为除五个食堂一个超市，还有学社联、学生会、校网站等一干机构坐落其中，更包括一个不错的湖南本地连锁书店和几个大小不一的会场。这些会场分别起名“立德厅”、“立言厅”和“立功厅”，通常，若非极为正式的场合需要动用硕大的科教大楼礼堂，一般的宣讲会、报告、文艺演出乃至每周的几部电影都以它们为场地。所以今天晚上19时许，当我在wonderfulmore的指引下，找到那张贴在通向立言厅的楼梯侧墙上“Adobe数字艺术之旅”的招贴时，也就想当然的以为上面写着的“场所未定”就应该在楼上三厅之一。

不过我错了，但这是后话。当时我爬上三楼，在结构古怪的走廊里左右寻摸。没有活动的时候这里光线很不好，只有校网站办公室等几个有人值班的门里透出几许光亮。立言厅锁着门，立德厅也黑着灯。于是又爬到四楼，在隐约传来的掌声和音乐声的指引下来到了立功厅。但这里显然不是什么数字之旅讲座，而是一场演出。会场里黑着灯，舞台上有四个流氓打扮的学生，分别执电琵琶、电弦子、电子琴和一架鼓，很专注地分别弄出些零碎的声响。我扫了一眼四周，没有看到什么标语或者banner来说明这是一场怎样的集会，正待开口问旁边的一个帅哥——这真是个帅哥，我准备问他的时候严重注意到了这一点——时，台上某个流氓把嘴贴在麦克风上如丧考妣的哼哼到：大家晚上好我们是XX乐队（没听清），很高兴冶金院邀请我们来参加这次元旦晚会，我们将为大家演唱两首歌，首先是酷玩乐队的Yellow。

客观的说他们唱得还不错，只是，不知道是因为紧张还是什么，四人一直都面无表情，直勾勾的分别盯着自己的手指拨片鼓槌和麦克风。在听到一半的时候我差点掉出窗外死掉：起先因为站得比较累我就想靠一会儿墙，但是在靠上去之前的一霎那一种异样的直觉阻止了我，转身掀开背后的窗帘，一扇窗台只及我大腿的窗户敞开着，四下根本没有什么可以应手抓住的东东。

一曲作罢，观众鼓掌，四人又在摆弄乐器的当儿，我和身边那个帅哥聊了几句。

——冶金院的？
——不，你呢？
——信息院的，来找Adobe数字之旅，没找到就进来看看。
——哦，我来看女主持的。
——女主持？
他向着台口一指。我望过去，一个身形娇美的女孩子站在那儿，双手捏着一只话筒。
——就为了来看一个美女？
——差不多吧，呵呵。
——你女朋友？
——不，我暗恋她。
我把注意力从那个倩影上转移到身边这个身高185公分上下，宽肩窄臀剑眉星目的GG身上来。有些人生来就是为某些成语作注脚的，属于他的成语大概应该是“玉树临风”。他脸上的表情并不复杂，如同刚才最后一句话一样相当平淡。很少有人会直接说“我暗恋某某某”，何况对一个陌生的猥琐男。据此我推测此人不喜欢婆婆妈妈，因为仔细想想这是说明何以仅仅为了不是女朋友的女主持人来看一场演出最合理而简短的解释。遇上了一仙人，我想。
——哦，暗恋多久了？
——两年。
我噎了一下，脑子里闪过漫画里众人跌倒两脚朝天的场景。
——一直都没跟她说？
——一直都没。
——太强了。
我脱口而出。他笑了笑，张了张嘴，但是什么都没说出来。我拍拍他的肩膀，绕过围在后场的观众，凑过去仔细看那个女主持。恰好此时乐队开始唱一首beyond的歌，她随之往舞台的方向前走了半步，并且很配合的撩了撩耳侧垂下的长发，向台下扫了一眼。她很漂亮，身材匀称，略有些瘦，淡淡画了妆，鼻子小巧，嘴巴也很精致，眼睛善良着，眉目间有点像周迅，穿一双靴子和连衣裙，披着一件大衣，静静而立，气质脱俗。台上的灯光将她所在的那个角落照得忽明忽暗，让人想起伊人灯火阑珊处之类的婉约词句。

作为数年前那场高校兼并风潮的结果，中南大学比较主要的校区有三个，彼此相距甚远。三个校区的学生也基本上各自为政，愿意的话完全可以老死不像往来。我们专业大学三年级从其中一个校区搬迁到现在的这个，导致我们这波大学三年级的家伙除了本系同学之外在校园里举目无亲，社团活动之类的也要重新来过。我的意思是，走在路上，我不再认识偶尔冒出来的美女了，而在原来那个校区，和路上的目光焦点打招呼，并听她们燕语莺声地叫声“小狐狸”，是对我乏味无比独身生活的一种美好调剂。好在目前这个校区姿源不足，今夜这个档次的mm如长沙夜空的星星一样稀少，我也就没有为此太过悲伤。不过，此刻看着面前这个显然是社团学生会成员之类的mm，我由衷希望自己还能像大二一样，可以直接或者间接的认识她，并且向她说起，有这么一个我凭第六感觉得值得一爱的GG在两年里暗恋着她。虽然谈恋爱是件麻烦的事情，虽然有一种可能是这将成为一场悲剧，虽然也许只有暗恋才是更美好的。

别提那个Adobe艺术之旅的讲座了。我在科教北楼南门找到了它的地点通告，旁边摆着某设计师证书培训机构的广告牌。如果那个粗制滥造的广告牌是这个培训点自己弄的，那么有很大的可能这所谓Adobe湖南之行第四场的讲座唯一的目的就是招徕学员。果然不出所料，在爬上六楼之后30分钟我溜了出来，同时对校本部艺术院（六楼是他们的地盘）也又产生了一丝煞有介事的怜悯。在这个寒冷的冬夜，这些有梦想和艺术细胞的年轻人们，坐在这间投影仪故障频频，音响效果极差的教室里，听一个大腹便便的中年人讲艺术系就业率是如何的低，而他们又是如何的需要"adaobi fotoxiaopu"。

亲爱的鬼树tracy tag 了我。第五次被tag，这让我觉得自己挺受欢迎的。不禁想要让那个女主持mm以及暗恋她的gg也来做一做，看一看愿望和现实之间相差多少，是不是足以让人心灰意冷，立地成佛。

题目：【选择伴侣的8个条件】

游戏规则:

  1. 必须注名你的理想伴侣是男的或者是女的
  2. 被点到玩游戏的人要写出自己选择伴侣的8个条件
  3. 要传给至少8个人一同参与这个游戏,并告知对方，被點名的人不可重覆被點，每個人只可玩一次

照例先说几句废话。首先，我觉得这个游戏基于一个假设，即我知道我选择伴侣的条件是什么。我果真知道么？我写下这些规则却和有了一个完全不一样的伴侣呢。其次，这个游戏做出了一种限定。如果写得不用心还则罢了，如果写得用心，以后就难免不去按图索骥。这样很容易无端的画地为牢，人的为关上了一些走向幸福的门。此外，规则A更确切的说法，应该是：“……理想伴侣是男and /or女的”。规则B和C，为什么是8个呢，8这个数字既不符合当今5个问题5个下线的国际标准，也不符合我认为7是最完美的数字这一审美情趣。Anyway，鉴于此游戏的种种瑕疵，以及我tag别人的结果通常都让我有一种挫败感，所以我打算做一次传销链的盲肠。

  1. 目前阶段，我可以确定你是个女人。
    1. 你应该是个感情真挚的人。怎样才算“感情真挚”，恕我保留它所指不明。
    2. 鉴于我如某些人的处女情结一样根深蒂固无可救药的美女情结，你得是个美女。请原谅我迷恋于此的肤浅，我会努力让自己值得你原谅，如果你是美女却真的不为此而自豪的话。
    3. 我希望你具有独立精神。这样，我们才能平等的对待对方，无论能相伴走多远，在心灵上都相敬如宾，而不是有什么征服或者附属的关系，不管谁对谁。
    4. 愿意与我分享欢乐，更重要的是愿意与我分享忧伤。但希望你不要看到那么多忧伤。如果需要一个类比，我希望你不是林黛玉，而是史湘云。而，在我的幽默感不幸弃我而去的时候，能够忍受我的无趣。抱歉我有时有些悲观。
    5. 你的语言和行为都能被我理解，如果我感到迷惑，你会解释。
    6. 爱我的家人，以及我的挚友。生活并不只有爱情。我希望它一部分是友情，最后变为亲情。
    7. 最理想而神圣的第七条：我希望你爱我。咋个爱法，请参考Oldimp的《我爱你》。
    8. 鉴于第七条的过于理想化，我只希望在我们互为伴侣的那段时间里，至少，你是愿意爱我的。
  3. 杨过，小龙女，凌波丽，陆雪琪，星雨者艾琳，醉梦，豪猪，刺猬。

尾注：通常有人问起想找个什么样滴女朋友之类滴问题，偶会这样回答：[善良，美丽，温柔，聪慧，依重要程度降序排列，要求至少满足包括美丽在内滴三条。]老实说，偶觉得目前偶还不配有一个符合上述标准滴女朋友，因为偶认为这样滴女孩子理应有个[正直，勇敢，体贴，智慧]滴男朋友。所以尽管遇到过若干颇为符合条件滴女生，偶都木有下手。偶知道这很愚蠢，只好自欺欺人说这是理智滴表现。就酱子。
