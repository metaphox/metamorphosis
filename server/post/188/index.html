什么标题好呢

寒潮过境，按键盘的手快冻僵了。刚刚从阴冷和昏暗中醒来，所做的梦还恍惚在眼前。上午无所事事的看了《美女缶》（日本电影，译为美女罐头。说实话我最开始只是冲着美女这两个字看它的，鄙视我吧。），下午就很应景的做出这个颇哲学的梦。我梦见我和醉梦坐在铁炉堡顶峰看夕阳，我对她说，也许我们以及我们关于以前的一切记忆，都是十分钟前被创造出来的。她说被谁创造出来的？我说可能用被字不是很确切，应该说我们以及我们关于以前的一切记忆都是在十分钟之前才出现的。十分钟之前什么都没有。她说，那可太可惜了，那不就是说，一切一切我们知道的过去发生过的美好的东西，其实都没有发生过。我说可是就算发生过也没有办法证明它发生过啊，真实只是我们的一种信念，凭我们个人的感知是没有办法确认任何已经发生的事情的。她说，即便真的是这样，我也愿意相信我所记忆的所有往事。我说，如果这一切也将在十分钟之后消失呢？

她未及作答，世界已经开始变得模糊。远处的地平线开始消退，峰峦的曲线逐渐出现锯齿，眼前的一切也从32万色变成256色。我惊恐的看到醉梦的身体开始碎裂，她的长发失去了魔法的保护，在越来越猛烈的风中飘散开来。似乎只过了一瞬间，我们所置身的世界已经破碎成一串串数据，在无数0和1之间，我向即将消失的醉梦说到，愿群星指引你。醉梦凄然一笑说，愿圣光照耀你。

最终一切归于沉寂。漆黑一片中有一行字冥冥浮现：服务器将在10分钟之后重新启动。

然后我醒了。我醒来的时候，这个梦仍旧是如此新鲜，以至于我可以清楚地记得醉梦最后一笑的表情。这大概是我几个月以来作的最有意思的一个梦，糅合了曾经读过的科幻小说，曾经结识的朋友和曾经困惑的哲学问题，从一个侧面证明我的创造力（捏造力，lol）尚未被消磨殆尽，只是找个地方藏起来了而已。

眼见约翰列农的忌日又要到了。去年这个时候，好像是Oldimp的指引，我访问了一个纪念他的西班牙语网站，在上面写了一段话，留了email。过了半个月左右，messenger上多了一个名叫Gustavo朋友，说看了我在网站的留言，同样作为John Lennon和Beatles的爱好者，想要和我聊聊。我就这样认识了这个打字飞快，除Beatles之外还喜欢看日本漫画尤其是DNA的墨西哥人。我们从寒假聊到暑假，后来他无端消失了很久。刚才端着咖啡看宿舍达达尼昂同学机器上Let it be的mtv，看着长发披肩的列侬和洋子，看着坐在钢琴后面络腮胡子西装革履的麦卡特尼，忽然就想起这个人来，遂爬上messenger，他居然在。

半年来我的蹩脚英语仍旧未见长进，所幸他仍像以前一样理解能力超群。他给我看了一张前两天去旅游时拍下的照片，说上面的一座山唤作Nayar Mountain，为的是纪年他们抗击西班牙征服者，名叫Nayar的民族英雄。于是在我贫瘠得要死的历史知识中，除Montezuma外又多了一个与古墨西哥有关的人名。说起Montezuma，在最近玩到的Sid"s Civilization IV中他被塑造成一个头戴长羽毛和骷髅头的Q版形象，是我第一场游戏中遇到的第一个邻国。莆一相见，他就语出惊人：让我们各杀100个奴隶，用他们的头来庆贺我们的友谊吧。

CIV4有一件挺讽刺的事情是它的一个bug。林林总总的十余个文明中，一些文明有两个可选君主。中国就是其中之一。雷打不动的一个当然是偶们的Mao，另一个则是秦始皇。但是，我选择秦始皇的时候却发现其头象分明是个胖头胖脑穿着鞑子衣服的家伙，而标明了成吉思汗的那位却是一身秦装。估计是开发人员将两人搞混了。So，就如同维大汉学系的教授不知道曹雪芹何许人也仍旧可以心安理得地做汉学教授一样，Sid的文明系列也一直秉承着歪曲别国的历史和特征的传统，（我又想起来在3代里日本武士的倭刀分明是龙泉宝剑的样式），可这并不影响它持续大卖。在这一点上，帝国时代三还是稍好一些。

一段摘录：

英国学者克里斯·罗杰克（Chris Rojek）在他那本《名流》中这样写道：“随着上帝的远去和教堂的衰败，人们寻求救赎的圣典道具被迫坏了。名人和奇观填补了空虚，进而造就了娱乐崇拜，同时也导致了一种浅薄、浮华的商品文化的统治。因而，娱乐崇拜掩饰了文化瓦解。商品文化无法造就完整的文化，因为它在每件商品上都打上了转瞬即逝和完全不可替代的烙印。同样，名流文化也无法产生卓越的价值，因为任何一种趋向卓越的努力都被商品化扼杀在摇篮中。”
